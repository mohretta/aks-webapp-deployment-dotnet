# AKS Web Application Deployment (.NET)

Déploiement d’une application web .NET sur Azure Kubernetes Service (AKS) avec conteneurisation Docker, pipeline CI dans Azure DevOps et gestion de plusieurs environnements de déploiement.

## Fonctionnalités
- Construction et publication d’une image Docker
- Déploiement sur Kubernetes (AKS)
- Gestion de plusieurs environnements (dev, test, prod)
- Configuration avec ConfigMaps et Secrets
- Exposition de l’application via Ingress
- Mise à l’échelle de l’application

## Technologies
- Azure DevOps
- Azure Kubernetes Service (AKS)
- Docker
- Kubernetes
- YAML
- .NET 8
- React
- Azure SQL Database

## Objectif
Mettre en place un flux DevOps complet pour déployer une application web conteneurisée dans un cluster Kubernetes Azure.
